## 编程语法

```js
export default {
	name: "MyComponent",
};
```

```html
<ul>
	<li v-for="todo in todos" :key="todo.id">{{ todo.text }}</li>
</ul>
```

```css
div {
	background: #fff;
}
```

### 语法高亮

1.多行：例如 {5-8}、{3-10}、{10-17}
2.多个单行：例如 {4,7,9}
3.多行与单行：例如 {4,7-13,16,23-27,40}
4.也可以使用 // [!code highlight] 注释实现行高亮。

```js{2}
export default {
    name:"MyComponent",
}
```

### 代码聚焦
```js
export default {
  data () {
    return {
      msg: 'Focused!' // [!code focus]
    }
  }
}
```
